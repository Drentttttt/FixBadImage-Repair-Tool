FixBadImage Repair Tool
=======================

Author: Adrein
Version: 1.0
Release Date: September 13, 2025
Platform: Windows 10/11

Overview:
---------
FixBadImage Repair Tool is a one-click utility designed to fix the Windows error 0xc000012f ("Bad Image"). It automates system repair tasks using trusted methods like DISM, SFC, and runtime installation — no technical knowledge required.

Features:
---------
- Runs with full admin elevation
- Repairs corrupted system files
- Installs missing Visual C++ Redistributables
- Logs all output to timestamped transcript files
- Optional restart prompt after repair

Usage:
------
1. Double-click FixBadImage.exe
2. Accept the admin prompt
3. Wait for the repair process to complete
4. Choose whether to restart your PC

Log files are saved to:  
`%TEMP%\FixBadImageLog_YYYY-MM-DD_HH-MM-SS.txt`

Requirements:
-------------
- Windows 10 or 11
- Administrator privileges
- Internet access (for runtime downloads)
